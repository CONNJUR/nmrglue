nmrglue.fileio.pipe.create_dic¶

nmrglue.fileio.pipe.create_dic(udic, datetimeobj=datetime.datetime(2012, 10, 5, 15, 41, 50, 41296))[source]¶

Crate a NMRPipe parameter dictionary from universal dictionary

This function does not update the dictionary keys that are unknown such as MIN/MAX, apodization and processing parameters, and sizes in none-current domain. Also rounding of parameter is different than NMRPipe.

Parameters :

udic : dict

Universal dictionary of spectral parameters.

datetimeobj : datetime object, optional

Datetime to record in NMRPipe dictionary
Returns :

dic : dict

Dictionary NMRPipe parameters.

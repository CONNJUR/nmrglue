nmrglue.fileio.sparky.write¶

nmrglue.fileio.sparky.write(filename, dic, data, overwrite=False)[source]¶

Write a Sparky file.

Parameters :

filename : str

Filename of Sparky file to write to.

dic : dict

Dictionary of Sparky parameters.

data : array_like

Array of NMR data.

overwrite : bool, optional

Set True to overwrite files, False will raise a Warning if the file exists.

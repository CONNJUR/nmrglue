nmrglue.process.proc_base.coad¶

nmrglue.process.proc_base.coad(data, clist, axis=-1)[source]¶

Coadd data.

Reduce data along axis by blocks data and multiplying by coefficients in clist. Incomplete blocks are discarded.

Parameters :

data : 2D ndarray

Array of NMR data data.

clist : list

List of Coefficients

axis : {0, 1, -1}

Axis to reduce. 0 for Y, 1 or -1 for X.

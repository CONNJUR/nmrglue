nmrglue.proc_base¶

A large collection of NMR spectral processing functions which operate on the last (-1) 2D arrays. These functions are wrapped by other processing modules. All parameter are assumed to be in units of points unless otherwise noted.

This module is imported as nmrglue.proc_base and can be called as such.

nmrglue.process.pipe_proc.qart¶

nmrglue.process.pipe_proc.qart(dic, data, a=0.0, f=0.0, auto=False)[source]¶

Scale Quad Artifacts

Parameters :

dic : dict

Dictionary of NMRPipe parameters.

data : ndarray

Array of NMR data.

a : float

Amplitude adjustment value.

f : float

Phase adjustment value.

auto : bool

True will perform a Gram-Schmidth orthorginalization to fund a and f automatically. Provided a and f parameters are ignored.
Returns :

ndic : dict

Dictionary of updated NMRPipe parameters.

ndata : ndarray

Array of NMR data with scaled quadrature artifacts.

Notes

Auto mode performs Gram-Schmidt orthogonalization, a different approach using a grid search is used in NMRPipe’s QART function.

nmrglue.analysis.linesh.unpack_table¶

nmrglue.analysis.linesh.unpack_table(rec, param_columns, amp_column)[source]¶

Unpack initial fitting parameters from a table.

Parameters :

rec : recarray

Records array (table) holding parameters.

param_columns : list

List of column names which hold lineshape parameters. See fit_NDregion() for format.

amp_column : str

Name of columns in rec holding initial amplitudes.
Returns :

params : list

List of initial parameter in the format required for fit_NDregion().

amps : list

List of initial peak amplitudes.
